Sesi 1: Pengantar OOP dan Dasar Kelas di Dart

Pendahuluan

Object Oriented Programming (OOP) merupakan paradigma pemrograman yang berfokus pada objek sebagai unit utama dengan data dan perilaku.

Konsep Dasar OOP

  • Kelas sebagai blueprint.
  • Objek sebagai instance.
  • Atribut sebagai data.
  • Metode sebagai perilaku.

Contoh Implementasi Kelas di Dart

class Car {
  String? model;
  String? color;

  Car(this.model, this.color);

  void start() {
    print('Mobil $model warna $color berjalan');
  }
}

void main() {
  Car myCar = Car('Sedan', 'Hijau');
  myCar.start();
}

Kesimpulan

Penerapan OOP pada bahasa Dart membantu pengembang membuat aplikasi yang lebih terstruktur, mudah dikembangkan, dan mudah dipelihara. Konsep ini sangat penting terutama dalam pengembangan aplikasi Flutter.
