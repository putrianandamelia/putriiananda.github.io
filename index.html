Pengantar OOP dan Dasar Kelas di Dart

Pendahuluan OOP

Object Oriented Programming (OOP) adalah paradigma pemrograman yang berfokus pada objek sebagai unit utama. OOP membuat kode lebih terstruktur, mudah dipelihara, dan dapat digunakan kembali.

Konsep Dasar: Kelas dan Objek

Kelas adalah blueprint atau cetakan, sedangkan objek adalah hasil nyata dari kelas tersebut. Kelas memiliki atribut dan metode.

Contoh Kode: Kelas dan Objek

class Car {
  String? model;
  String? color;

  void start() {
    print('Car started!');
  }

  void stop() {
    print('Car stopped!');
  }
}

void main() {
  Car myCar = Car();
  myCar.model = "Sedan";
  myCar.color = "Merah";
  myCar.start();
  myCar.stop();
}

Ringkasan

  • OOP berfokus pada objek dan interaksi.
  • Kelas adalah blueprint, objek adalah instance.
  • Constructor digunakan untuk inisialisasi data.
