1. Pengertian OOP

Object Oriented Programming (OOP) adalah paradigma pemrograman yang berorientasi pada objek. Objek dibuat dari sebuah kelas yang berisi atribut dan method.

  • Class dan Object
  • Encapsulation
  • Constructor
  • Method

2. Class dan Object


class Car {
  String? model;
  String? color;

  void start() {
    print('Car started!');
  }
}

3. Constructor


Car(this.model, this.color);

Constructor digunakan untuk menginisialisasi nilai atribut.
