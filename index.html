Pengertian OOP

Object Oriented Programming (OOP) adalah paradigma atau metode pemrograman yang berfokus pada objek sebagai komponen utama dalam membangun sebuah program. Dalam OOP, program dibangun dari objek-objek yang memiliki data (state) dan perilaku (behavior).

Objek, Kelas, dan Konsep Dasar

Kelas adalah cetakan untuk membuat objek. Objek adalah hasil dari kelas yang memiliki atribut (data) dan metode (fungsi).

Class Sederhana (Car)

Contoh struktur dasar class di Dart.

DART // RUNTIME
class Car {
  String? model;
  String? color;

  void start() {
    print('Car started!');
  }
}

Membuat Objek dari Kelas

Membuat instance dari class agar dapat digunakan dalam program.

DART // RUNTIME
void main() {
  Car myCar = Car();
  myCar.model = "Sedan";
  myCar.color = "Merah";

  myCar.start();
}

Constructors

Constructor digunakan untuk menginisialisasi nilai awal saat objek dibuat.

DART // RUNTIME
class Car {
  String? model;
  String? color;

  Car(this.model, this.color);
}

Latihan: Class Mahasiswa

DART // RUNTIME
class Mahasiswa {
  String nama;
  String nim;
  double ipk;
}

Encapsulation

DART // RUNTIME
class Person {
  String _name;
  int _age;
}

This Keyword

DART // RUNTIME
class Product {
  String nama;
  Product(this.nama);
}

Latihan: Class Produk

DART // RUNTIME
class Produk {
  final String id;
  final String nama;
  final double harga;
}

Kesimpulan

OOP adalah metode pemrograman yang berfokus pada objek yang memiliki data dan fungsi. Konsep seperti class, object, constructor, dan this membantu membuat program lebih terstruktur dan mudah dikembangkan.
