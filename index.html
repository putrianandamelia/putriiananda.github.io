Pengertian OOP

Object Oriented Programming (OOP) adalah paradigma atau metode pemrograman yang berfokus pada objek sebagai komponen utama dalam membangun sebuah program. Dalam OOP, program tidak hanya terdiri dari kumpulan instruksi, tetapi dibangun dari objek-objek yang memiliki data (state) dan perilaku (behavior). Konsep ini meniru dunia nyata, di mana setiap benda memiliki karakteristik dan fungsi tertentu. OOP bertujuan untuk membuat kode lebih terstruktur, mudah dipahami, mudah dikembangkan, serta dapat digunakan kembali.

Object, Class, dan Konsep Dasar

Class adalah cetakan atau rancangan untuk membuat object. Object adalah hasil dari class yang memiliki atribut (data) dan metode (fungsi). class menentukan bagaimana object dibuat dan bagaimana object tersebut bekerja.

Class Sederhana (Car)

Aturan penulisan struktur class di Dart terdiri dari atribut dan metode. Penulisannya diawali dengan kata kunci class, kemudian nama kelas dan isi kelas ditulis di dalam tanda kurung kurawal { }. berikut contoh nya. 

// Nama  : Putri Ananda Amelia

  String? model;
  String? color;

  void start() {
    print('Car started!');
  }
}

Membuat Object dari Class

Membuat object adalah proses menghasilkan instance dari sebuah class agar dapat digunakan dalam program. Setelah object dibuat, atribut dan metode dalam class dapat diakses melalui object tersebut. 

// Nama  : Putri Ananda Amelia
// NIM   : 251410029
// Kelas : SI2A
  
void main() {
  Car myCar = Car();
  myCar.model = "Sedan";
  myCar.color = "Merah";

  myCar.start();
}

Constructors

Constructor adalah metode khusus dalam class yang digunakan untuk menginisialisasi atau memberikan nilai awal pada object saat object dibuat. 

// Nama  : Putri Ananda Amelia
// NIM   : 251410029
// Kelas : SI2A

  String? model;
  String? color;

  Car(this.model, this.color);
}

Encapsulation

Encapsulation adalah konsep OOP yang membungkus data (atribut) dan method dalam satu class serta membatasi akses langsung ke data tersebut. Tujuannya untuk menjaga keamanan data agar tidak bisa diubah sembarangan dari luar class. Biasanya digunakan dengan membuat atribut menjadi private dan mengaksesnya melalui getter dan setter. 

// Nama  : Putri Ananda Amelia
// NIM   : 251410029
// Kelas : SI2A
  
class Person {
  String _name;
  int _age;
}

This Keyword

this adalah kata kunci yang digunakan untuk merujuk pada object atau instance yang sedang aktif di dalam class.

Kesimpulan

OOP adalah metode pemrograman yang berfokus pada object yang memiliki data dan fungsi. Dalam Dart, konsep seperti class, object, constructor, dan this digunakan untuk membangun program yang lebih terstruktur dan mudah dikembangkan. Pemahaman dasar OOP sangat penting sebagai fondasi dalam pembuatan aplikasi.
